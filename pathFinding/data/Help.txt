ИНСТРУКЦИЯ К ПРОГРАММЕ.
Данная программа предназначения для нахождения пути с наименьшей стоимостью при помощи двух алгоритмов: Дейкстры и A star.
После запуска приложения в основном меню можно сделать следущее:
-Ввести данные
-Найти решение

Ввод данных осуществляется двумя способами:
--Ввод с консоли. 
	Введите в нужном формате размер карты, количество стен, их координаты, а также координаты начальной и конечной точек. Для удобства ввода координат стенок в консоле отобразится карта нужного размера.
--Загрузка данных из json-файла.
	При выборе данного пункта меню будет отображен список файлов, доступных для чтения. При помощи стрелок выберите один из файлов. Данные файлы находятся в проекте в папке data/input_data

После ввода данных будет доступно меню поиска решений. Здесь можно выбрать тот или иной алгоритм для решения карты, а также сохранить результат в файл c расширением .txt. Решения записываются в папку data/results. При выборе того или иного алгоритма для решения, на консоли будет отображена карта с отмеченными начальными и конечными точками, а также фиолетовым цветом обозначен путь.
